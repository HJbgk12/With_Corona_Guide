# With_Corona_Guide

위드 코로나를 극복하기 위한 사용자 맞춤형 위치 기반 다중이용시설 가이드 웹 서비스 프로젝트입니다.

# 목적

- 주기적으로 변화하는 거리두기 규율로 국민들이 혼란을 겪는 문제 발생
- 위드코로나에 맞추어 백신 접종 여부에 따른 이용시설 규제 정보 파악 필요

## 프레임워크

Django
- 쉽고 빠른 개발
- 풍부한 라이브러리 : 데이터 파싱, 분석, API 연동 등 활용 가능한 라이브러리 多
